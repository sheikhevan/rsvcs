# RSVCS (Rust Version Control System)
## What is this?
This program is a simple version control system that runs locally on your machine. In reality, it is less a vcs and more a backup utility.
It allows you to define both the directory files will be backed up to and which files will be backed up. The configuration file is written
in [YAML](https://yaml.org/).

## How should I use this?
I would add it to a cron job.
